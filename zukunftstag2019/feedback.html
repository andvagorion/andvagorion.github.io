Radio-Button

Ein Radio-Button ermöglicht eine Auswahl zwischen verschiedenen Werten.

Das name-Attribut der Radio-Buttons gibt an, zu welcher Auswahl die Buttons gehören.


				<input name="note-mathe" type="radio" value="1">
				<input name="note-mathe" type="radio" value="2">
				<input name="note-mathe" type="radio" value="3">
				<input name="note-mathe" type="radio" value="4">
				<input name="note-mathe" type="radio" value="5">
				<input name="note-mathe" type="radio" value="6">

Input-Label

Zu einem Input-Element kann ein Label erstellt werden, über das das Input-Element ausgewählt werden kann. Die Attribute for und id müssen dabei gleich sein. 


				<label for="cat">Katze</label>
				<input id="cat" type="radio" name="favorite-animal" value="cat">

Tabellen

Mit Hilfe des table-Elements kann eine Tabelle erzeugt werden.

Der Kopfteil der Tabelle wird durch thead markiert, der Körperteil durch tbody

Jede Zeile / row wird dabei durch ein tr-Element repräsentiert.

Kopfzellen sind th-Elemente, Körperzellen sind td-Elemente.


<table>
  <thead>
    <tr>
      <th>Tier</th>
      <th>Name</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Hund</td>
      <td>Bello</td>
    </tr>
    <tr>
      <td>Katze</td>
      <td>Felicitas</td>
    </tr>
  </tbody>
</table>
