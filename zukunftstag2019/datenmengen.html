JSON-Daten

Die Filme liegen als Datensätze in einer JSON-Datei vor:

  • JSON-Datei

Aufgaben

  • Wie viele Filme befinden sich in der Datenbank? Tipps
    • Google: "Länge JavaScript Array"
  • Auflistung aller Filmtitel Tipps
    • Loop über alle Filme
    • Ausgabe in der Konsole: console.log(...)
  • Aus welchem Jahr ist der neueste Film? Tipps
    • Variable year, um Jahr zu speichern erstellen: var year = 0
    • Loop über alle Filme
    • Prüfen, ob Jahr des aktuellen Films größer ist als das in year gespeicherte Jahr
    • Falls ja: Variable year mit dem neuen Jahr setzen
  • In wie vielen Filmen spielt Nicolas Cage mit? Tipps
    • Erstellen einer Zähl-Variablen: var count = 0
    • Loop über alle Filme
    • Schauspieler*innen sind im Attribut cast aufgelistet: movie["cast"]
    • Prüfen, ob Nicolas Cage mitspielt: String.indexOf
    • Falls ja: Zähl-Variable hochzählen
  • Welche Genres werden mehr als 100 Mal verwendet? Tipps
    • Erstellen eines Objekts, das die verwendete Anzahl eines Genres hält: var genreCount = {}
    • Loop über alle Filme
    • Genres sind für jeden Film komma-getrennt
    • Aufsplitten eines Strings: Array.split

Schema

Die Filme sind in einem Array ([]) mit Objekten ({}) gespeichert.


							[{
								"title": "127 Hours",
								"year": 2010,
								"director": "Danny Boyle",
								"cast": "James Franco, Amber Tamblyn, Kate Mara, Clémence Poésy, Lizzy Caplan",
								"genre": "Biography, Drama"
							},
							{
								...
							}]

Laden der Daten


							jQuery.getJSON("https://raw.githubusercontent.com/andvagorion/andvagorion.github.io/master/movies.json", function(movies) {
								console.log(movies);
								...
							});

Arbeiten mit Objekten

Loop: 

							for (var key in movies) {
								var movie = data[key];

								console.log(movie.title);
								// oder
								console.log(movie["title"]);
							}
